# Tic Tac Toe

**作品名稱：** 闖關遊戲
	
**動機：** 此作品動念於想練習MinMax演算法，而興起該遊戲製作。

**開發環境：** Phaser(主要是運用Javascript進行開發)

**遊戲解說：** 進行遊戲過程，必須避開敵人，並吃足夠金幣，直到地圖的最後，方可闖關成功。
