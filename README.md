# Games developed based on Phaser

**作品名稱：** 類瑪利歐的闖關遊戲
	
**動機：** 此作品主要致敬過往孩時，曾熱衷Platformer Games，也好奇製作方式，而興起該遊戲製作。

**開發環境：** Phaser(主要是運用Javascript進行開發)

**遊戲解說：** 進行遊戲過程，必須避開敵人，並吃足夠金幣，直到地圖的尾聲，方可闖關成功。
